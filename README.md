# FastAPI Best Architecture UI

Front-end Implementation of the [FastAPI Best Architecture](https://github.com/fastapi-practices/fastapi_best_architecture)

## Run

```shell
npm install
npm run dev
```

or

```shell
yarn install
yarn run dev
```
